# Project-RGB-Glasses
2017年度高専祭情報工学科展出展
フルカラーLEDを使ったメガネのイルミネーションパターンを設定するGUIアプリケーション
イルミネーションパターンはバイナリ形式でSDカード保存し，Arduino側でSDカードからイルミネーションパターンを読み込む

# 使用技術
- Arduino
- Java (Swing)

## イルミネーションパターン
1. コマ送り
2. 電光掲示板（左右）
3. 電光掲示板（上下）
その他に，特定のイルミネーションパターンを再生後，別のイルミネーションパターンに自動遷移するようにも設定することができる．
